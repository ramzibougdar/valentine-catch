# Valentine's Day Proposal Page

A beautiful, interactive, and personalized web page for a Valentine's Day proposal.

## Features

- **Personalized Experience**: Use URL parameters to customize the page.
  - `name`: Prepend a name to the proposal title (e.g., `?name=Alex`).
  - `lang`: Switch between English (`en`) and French (`fr`) (e.g., `?lang=fr`).
- **Interactive "No" Button**: The "No" button playfully evades the mouse cursor within the container, avoiding the "Yes" button.
- **Heart Explosion**: A celebratory heart explosion animation when the "Yes" button is clicked.
- **Responsive Design**: Works on mobile and desktop devices.
- **Modern Aesthetics**: Glassmorphism, smooth gradients, and elegant typography.


### Customization Examples

- **English**: `index.html?name=Valentine`
- **French**: `index.html?name=Valentine&lang=fr`

## Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom styling with variables and animations.
- **Vanilla JavaScript**: Interactive logic and URL parameter handling.
